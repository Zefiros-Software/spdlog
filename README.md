#Fmt
To use [Fmt](https://github.com/fmtlib/fmt) in [ZPM](http://zpm.zefiros.eu), just use:

# [ZPM](http://zpm.zefiros.eu) Installation
In `.package.json`
```json
"requires": [
    {
        "name": "Zefiros-Software/Fmt",
        "version": "^3.0.0"
    }
]
```

In `zpm.lua`
```lua
zpm.uses "Zefiros-Software/Fmt"
```
